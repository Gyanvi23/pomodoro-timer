# 🍅 Pomodoro Timer

A simple and interactive **Pomodoro Timer Web App** designed to help you stay focused and productive using the Pomodoro Technique.

---

## 🚀 Live Demo

🔗 [View Project](https://pomodorotimer00.netlify.app/)

---

## 📌 Features

* ⏱️ 25-minute work timer & 5-minute break timer
* 🔄 Automatic switching between work & break sessions
* 📊 Progress ring animation
* 🌙 Dark mode toggle
* 🎨 Multiple themes (Default, Dark, Sunny, Forest, Galaxy + Custom color)
* 🎵 Background music options (Piano, Guitar, Nature sounds)
* 🔔 Sound alert when session ends
* 📈 Daily Pomodoro counter (stored using localStorage)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
├── index.html
```

All the code (HTML, CSS, JS) is written in a single file.

---

## ⚙️ How It Works

* Timer starts with **25 minutes (Work Mode)**
* After completion → switches to **5-minute Break Mode**
* Pomodoro count increases after each completed work session
* Data is saved in browser using **localStorage**

---

## 💡 Key Functionalities

* Start / Pause / Reset timer
* Theme selection & customization
* Background music control
* Persistent data (theme + count)
